# 퇴근구름 개인정보 처리방침

[(주)비트리더](이하 "회사")은(는) 「개인정보 보호법」 제30조에 따라 정보주체의 개인정보를 보호하고
이와 관련한 고충을 신속하고 원활하게 처리할 수 있도록 다음과 같이 개인정보 처리방침을 수립·공개합니다.

> **요약**: 퇴근구름은(는) 사용자 PC에서 동작하는 단독형(클라이언트) 데스크톱 앱입니다.
> 회사는 별도의 회원가입·로그인·계정을 운영하지 않으며, 사용자를 식별할 수 있는 정보를
> 회사 서버로 수집·전송·보관하지 않습니다. 모든 설정값과 입력 정보는 **사용자 PC 내부에만**
> 저장됩니다. 날씨 조회에 필요한 정보(위치·인증키)는 사용자가 선택한 외부 날씨 서비스로만 전송됩니다.

---

## 1. 개인정보의 처리 목적

회사는 다음의 목적을 위하여 최소한의 개인정보를 처리하며, 처리하는 개인정보는 다음 목적 이외의
용도로는 이용하지 않습니다.

- 사용자가 설정한 지역의 **날씨 정보 조회 및 표시**
- 근무 진행률·오버레이 등 **앱 기능 제공 및 사용자 설정 유지**
- 외부 날씨 서비스 연동을 위한 **API 인증키 보관 및 사용**

## 2. 처리하는 개인정보의 항목

회사는 앱 기능 제공을 위해 다음 정보를 사용자 PC에서 처리합니다. 이 정보들은 회사 서버로
수집되지 않으며, 사용자 단말에만 저장됩니다.

| 구분 | 항목 | 수집 방법 | 필수/선택 |
|---|---|---|---|
| 위치 정보 | 도시명, 위·경도 좌표 또는 행정구역명(시도/시군구/읍면동·기상청 격자) | 사용자 직접 입력, 또는 "현재 지역 검색" 시 Windows 위치 서비스 | 필수(앱 기능상) |
| 인증 정보 | 사용자가 발급받은 날씨 서비스 API 인증키 | 사용자 직접 입력 | 선택 |
| 앱 설정 | 근무 시작/종료 시각, 표시 요일, 화면 테마·언어, 오버레이 디자인 등 | 사용자 직접 설정 | 선택 |

- 회사는 **이름·전화번호·이메일·주민등록번호 등 직접 식별정보**, **고유식별정보**, **민감정보**,
  **계정·로그인 정보**, **기기 식별자**를 수집하지 않습니다.
- 회사는 만 14세 미만 아동의 개인정보를 별도로 수집·처리하지 않습니다.
- 위치 정보는 날씨 조회 목적에 필요한 최소한으로만 처리됩니다.

## 3. 개인정보의 처리 및 보유 기간

- 위 모든 정보는 **사용자 PC 내부에만 저장**되며, 회사가 별도로 보관하는 정보는 없습니다.
- 정보는 사용자가 앱에서 직접 변경·삭제하거나, **앱을 제거할 때까지** 사용자 단말에 보관됩니다.
- 설정 초기화 또는 인증키 삭제 기능을 사용하면 해당 정보는 단말에서 즉시 삭제됩니다.

## 4. 개인정보의 파기 절차 및 방법

- **파기 절차**: 보유 목적이 달성되거나 사용자가 삭제를 요청·실행하면 해당 정보를 파기합니다.
- **파기 방법**
  - 설정 파일(예: `settings.json`)에 저장된 정보: 설정 초기화 또는 앱 제거 시 파일과 함께 삭제
  - 인증키 등 보안 저장 정보: 앱의 "삭제" 기능 또는 OS 자격증명 저장소에서 삭제
  - 전자적 파일은 복구·재생이 불가능한 방법으로 삭제됩니다.
- 회사 서버에 보관하는 정보가 없으므로 서버 측 파기 대상은 존재하지 않습니다.

## 5. 개인정보의 제3자 제공

회사는 정보주체의 개인정보를 제3자에게 제공하지 않습니다.

다만 사용자가 날씨 조회 기능을 사용하면, 조회에 필요한 정보가 **사용자가 직접 선택한 외부 날씨
서비스**로 전송됩니다. 이는 회사가 제공하는 것이 아니라 사용자의 요청에 따라 해당 서비스를
이용하는 과정에서 발생합니다.

| 전송 대상 | 전송되는 정보 | 목적 |
|---|---|---|
| WeatherAPI.com (선택 시) | 도시명, 표시 언어 코드, 사용자 API 키 | 날씨 정보 조회 |
| 기상청 단기예보(공공데이터포털, data.go.kr) (선택 시) | 기상청 격자 좌표(nx, ny), 사용자 API 키 | 국내 날씨 정보 조회 |

- 회사는 위 외부 서비스에 사용자를 식별할 수 있는 정보를 전송하지 않습니다.
- 외부 서비스의 개인정보 처리에 대해서는 각 서비스의 개인정보 처리방침이 적용됩니다.

## 6. 개인정보 처리업무의 위탁

회사는 개인정보 처리업무를 외부에 위탁하지 않습니다.

## 7. 개인정보의 국외 이전

날씨 제공자로 **WeatherAPI.com**을 선택한 경우, 날씨 조회를 위해 도시명·언어 코드 및 사용자
API 키가 해외에 소재한 해당 서비스로 전송될 수 있습니다.

| 이전받는 자 | 이전 국가 | 이전 항목 | 이전 일시 및 방법 | 이용 목적 |
|---|---|---|---|---|
| WeatherAPI.com 운영사 | 국외(서비스 운영 국가) | 도시명, 언어 코드, 사용자 API 키 | 날씨 조회 시점에 HTTPS 통신으로 전송 | 날씨 정보 조회 |

- 기상청(공공데이터포털) 제공자를 선택한 경우 정보는 국내로만 전송됩니다.
- 사용자는 날씨 제공자를 변경하거나 기능을 사용하지 않음으로써 국외 이전을 거부할 수 있습니다.

## 8. 개인정보의 안전성 확보조치

회사는 처리하는 정보의 안전성을 확보하기 위해 다음 조치를 시행하고 있습니다.

- **수집 최소화 및 로컬 보관**: 앱 기능에 필요한 최소한의 정보만 처리하며, 회사 서버로 수집·전송하지 않습니다.
- **인증키 보호**: 날씨 서비스 API 인증키는 평문으로 노출되지 않도록 운영체제의 자격증명
  저장소(Windows 자격 증명 관리자) 또는 사용자 계정 단위 암호화(DPAPI)로 보호하여 저장합니다.
- **전송 구간 보호**: 외부 날씨 서비스와의 통신은 가능한 경우 HTTPS를 사용합니다.
- **접근 통제**: 저장된 정보는 해당 PC 사용자 계정 권한 범위 내에서만 접근됩니다.

## 9. 개인정보 자동 수집 장치(쿠키 등)의 설치·운영 및 거부

퇴근구름은(는) 쿠키 등 인터넷 접속정보를 자동으로 수집하는 장치를 설치·운영하지 않습니다.
또한 행태정보 수집, 맞춤형 광고, 분석·통계(텔레메트리) 도구를 사용하지 않습니다.

## 10. 정보주체와 법정대리인의 권리·의무 및 행사 방법

- 정보주체는 자신의 정보에 대해 언제든지 **열람·정정·삭제·처리정지**를 요구할 수 있습니다.
- 본 앱의 정보는 사용자 PC에 저장되므로, 사용자는 앱의 **설정 화면에서 직접 조회·수정**하거나,
  **설정 초기화·인증키 삭제·앱 제거**를 통해 직접 삭제·처리정지를 수행할 수 있습니다.
- 권리 행사에 관해 문의가 필요한 경우 아래 개인정보 보호책임자에게 연락할 수 있으며, 회사는
  지체 없이 안내·조치합니다.
